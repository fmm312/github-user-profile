  - Repositories filters
  - **Responsive design**
  - CD at **Vercel**

## Getting started

Clone this repository and install its dependencies:

```sh
$ git clone https://github.com/fmm312/github-user-profile
$ cd github-user-profile
$ yarn
$ yarn start
```
